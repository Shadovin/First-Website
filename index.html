Languages

HTML

HTML is a language which creates structure of your website. If your dream is to work in Front-End, HTML will be your first step to obtain it.

CSS

CSS is a language used to style your website. It's the second step to start working in Front-End. By using CSS you can change many things such as: font size, background or size of the images. I think it's more difficult than HTML or CSS, but if you spend a lot of time on it, it will be simple for you.

Java Script

Java Script demands analitycal thinking.It's programing language, by using it you may do some awesome things. For example you can write your own programe, like "To do list" or something more creative. For me, learning it was very demanding, but everything is possible if you only want.
